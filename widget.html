TinyG Workspace

chilipeppr.com/tinyg

This workspace is a full-blown Gcode sender for your TinyG. The workspace connects to your TinyG via the serial port over the ChiliPeppr Serial Port JSON Server. It lets you view your Gcode in 3D, view the XYZA positions, perform jogging, and see advanced real-time details from the TinyG.

Grbl Workspace

chilipeppr.com/grbl

Grbl devices are now supported inside ChiliPeppr. The workspace connects to your Arduino via the serial port over the ChiliPeppr Serial Port JSON Server. Maintained by Jarret Luft.

Serial Port Workspace

chilipeppr.com/serialport

This is a workspace for a generic Serial Port console for ChiliPeppr's Hardware Fiddle. This workspace depends on ChiliPeppr's Serial Port JSON Server.

Terminal Workspace

chilipeppr.com/terminal

This workspace lets you interact with the command line of the host running the Serial Port JSON Server. Useful for interacting with your host hardware like the Raspberry Pi from Javascript.

Popular Workspace User Contributions

TinyG Tablet Workspace

chilipeppr.com/tinygtablet

This workspace is ideal for your tablet so you can jog around your TinyG CNC machine. It is just the main TinyG workspace with alternate CSS for a better tablet experience.

Grbl Raspicam

chilipeppr.com/grbl_raspicam

Let's you watch your Raspicam inside an alternate version of the Grbl Workspace.

Xpix Eagle PCB Workspace

chilipeppr.com/xpix

Created by Frank Herrmann, this is an advanced workspace geared towards extending the Eagle BRD Import widget to support better milling, drilling, solder paste dispensing, and pick and place.

Tigershark Workspace

chilipeppr.com/tigershark3d

This workspace is geared towards 3D printers and the Tigershark 3D printing control board which is based on TinyG.
